# STM32F103C8T6 双UART数据转发器技术规格

## 1. 硬件规格

### 1.1 微控制器规格
| 参数 | 值 | 说明 |
|------|----|----|
| 型号 | STM32F103C8T6 | ARM Cortex-M3内核 |
| 封装 | LQFP48 | 48引脚封装 |
| Flash | 64KB | 程序存储器 |
| RAM | 20KB | 数据存储器 |
| 工作电压 | 2.0V - 3.6V | 标准3.3V供电 |
| 工作温度 | -40°C ~ +85°C | 工业级温度范围 |

### 1.2 时钟配置
| 时钟源 | 频率 | 配置 |
|--------|------|------|
| HSE | 8MHz | 外部晶振 |
| PLL | 72MHz (设计值) | HSE × 9 |
| SYSCLK | 8MHz (当前) | 系统时钟 |
| HCLK | 8MHz | AHB总线时钟 |
| PCLK1 | 4MHz | APB1总线时钟 (÷2) |
| PCLK2 | 8MHz | APB2总线时钟 |

### 1.3 引脚分配
| 引脚 | 功能 | 配置 | 说明 |
|------|------|------|------|
| PA2 | USART2_TX | 推挽输出 | 发送到上位机 |
| PA3 | USART2_RX | 浮空输入 | 接收上位机命令 |
| PA9 | USART1_TX | 推挽输出，高速 | 发送到H750 |
| PA10 | USART1_RX | 浮空输入 | 接收H750数据 |
| PA13 | SWD-IO | 调试接口 | 程序下载/调试 |
| PA14 | SWD-CLK | 调试接口 | 调试时钟 |
| PD0 | OSC_IN | HSE晶振输入 | 8MHz |
| PD1 | OSC_OUT | HSE晶振输出 | 8MHz |

## 2. 通信规格

### 2.1 UART配置
| 参数 | UART1 | UART2 |
|------|-------|-------|
| 波特率 | 921600 bps | 256000 bps |
| 数据位 | 8位 | 8位 |
| 停止位 | 1位 | 1位 |
| 校验位 | 无 | 无 |
| 流控 | 无 | 无 |
| 模式 | 全双工 | 全双工 |

### 2.2 DMA配置
| DMA通道 | 用途 | 优先级 | 模式 | 缓冲区大小 |
|---------|------|--------|------|------------|
| DMA1_Ch4 | UART1_TX | 中等 | 正常 | 3648字节 |
| DMA1_Ch5 | UART1_RX | 高 | 正常 | 3648字节 |
| DMA1_Ch6 | UART2_RX | 低 | 正常 | 1字节 |
| DMA1_Ch7 | UART2_TX | 低 | 正常 | 可变 |

## 3. 协议规格

### 3.1 命令格式
| 命令字 | 功能 | 数据长度 | 响应时间 |
|--------|------|----------|----------|
| 0xA1 | 原始数据模式 | 变长 | < 100ms |
| 0xA2 | 8位AD数据模式 | 3648字节 | < 100ms |
| 0xA4 | 处理数据模式 | 729字节 | < 100ms |
| 0xB1 | 曝光时间级别1 | 变长 | < 50ms |
| 0xB2 | 曝光时间级别2 | 变长 | < 50ms |
| ... | ... | ... | ... |
| 0xBF | 曝光时间级别15 | 变长 | < 50ms |

### 3.2 数据包结构
```
命令包 (上位机 → STM32 → H750):
+--------+
| CMD    | 1字节命令字
+--------+

数据包 (H750 → STM32 → 上位机):
+--------+--------+--------+--------+
| DATA0  | DATA1  | ...    | DATAn  | n字节数据
+--------+--------+--------+--------+
```
