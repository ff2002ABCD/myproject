# FD-MGS-A 光谱分析系统

## 📊 系统概述

FD-MGS-A 光谱分析系统是专为STM32光谱仪设计的专业数据采集和分析软件。系统支持3648像素点的高精度光谱数据采集，具备完整的光谱标定功能和实时数据分析能力。

### 🎯 主要特点

- **高精度数据采集**: 支持3648像素点光谱数据，16进制格式(00-FF)
- **专业坐标系统**: 像素位坐标(0-3648，500一档) + 光强度坐标(0-60000，10000一档)
- **光谱标定功能**: 6种标准汞灯峰位，线性拟合算法，自动计算标定参数
- **实时数据显示**: 基于matplotlib的专业光谱图绘制
- **多种采集模式**: 单次采集、连续采集、可调间隔
- **积分时间控制**: 15种积分时间选项(10us-500ms)
- **数据管理**: 光谱数据保存/加载、标定结果导出

数据规格:

## 🔧 使用指南

### 1. 基础操作流程

#### 步骤1: 连接设备
1. 连接STM32光谱仪到电脑USB端口
2. 在软件中选择正确的串口
3. 设置波特率(推荐115200或更高)
4. 点击"连接"按钮

#### 步骤2: 设置积分时间
1. 从下拉菜单选择积分时间(10us-500ms)
2. 点击"设置"按钮发送命令
3. 确认积分时间设置成功

#### 步骤3: 采集光谱数据
```
单次采集:
- 点击"单次采集"按钮
- 发送0xA2命令
- 等待3648字节数据接收

连续采集:
- 设置采集间隔(50-10000ms)
- 点击"连续采集"按钮
- 定时自动重复采集
- 点击"停止采集"结束
```

### 2. 光谱标定操作

#### 进入标定模式
1. 点击"开始标定"按钮
2. 系统进入峰位选择状态
3. 准备添加标定点

#### 添加标定点
1. 在光谱图上观察峰位
2. 从下拉菜单选择对应的标准峰
3. 输入峰位对应的像素位置
4. 点击"添加"按钮

#### 完成标定
1. 至少添加2个标定点(推荐4-6个)
2. 点击"导入标定"按钮
3. 系统自动计算拟合参数
4. 显示标定结果: k, b, r²
5. 光谱图自动切换到波长显示

### 3. 数据管理

#### 保存光谱数据
- 菜单: 文件 → 保存光谱数据
- 格式: CSV文件
- 内容: 像素位/波长 + 光强度

#### 加载历史数据
- 菜单: 文件 → 加载光谱数据
- 支持CSV格式的光谱文件
- 自动更新图表显示

#### 导出标定结果
- 菜单: 文件 → 导出标定结果
- 格式: JSON文件
- 内容: k, b, r², 标定点, 时间戳

## 🔬 技术详解

### 1. 数据传输协议

#### 命令格式
```
积分时间设置:
0xB1-0xBF (对应15种积分时间)

光谱采集:
0xA2 (采集高8位ADC数据)
```

#### 数据格式
```
传输数据: 3648字节
数据类型: 无符号8位整数 (0x00-0xFF)
像素排列: 按像素位置顺序
数据含义: 每字节代表一个像素的光强度
```

### 2. 光谱标定算法

#### 数学模型
```
线性拟合方程: λ = k·x + b

其中:
λ: 波长 (nm)
x: 像素位置 (0-3647)
k: 标定系数 (nm/pixel)
b: 标定截距 (nm)
```

#### 计算方法
```python
# 使用numpy.polyfit进行线性拟合
coefficients = np.polyfit(pixels, wavelengths, 1)
k = coefficients[0]  # 标定系数（斜率）
b = coefficients[1]  # 标定截距

# 计算决定系数 r²
correlation = np.corrcoef(pixels, wavelengths)[0, 1]
r_squared = correlation ** 2
```

#### 质量评估
```
r² > 0.99: 优秀拟合
r² > 0.95: 良好拟合
r² < 0.95: 需要改进
```

### 3. 图表显示系统

X轴(波长模式):

## 🔍 故障排除

### 常见问题

#### 1. 串口连接失败
```
问题: "串口连接失败"
解决方案:
- 检查设备连接
- 确认串口号正确
- 检查其他程序是否占用串口
- 尝试不同波特率
```

#### 2. 数据长度错误
```
问题: "数据长度错误: xxx/3648"
解决方案:
- 检查STM32端数据发送逻辑
- 确认发送3648字节完整数据
- 检查串口传输稳定性
- 调整波特率和传输间隔
```

#### 3. 标定拟合失败
```
问题: "标定计算失败"
解决方案:
- 确保至少有2个标定点
- 检查标定点数据合理性
- 避免重复的像素位置
- 选择分布较广的标定点
```

#### 4. 图表显示异常
```
问题: 光谱图显示错误
解决方案:
- 检查matplotlib依赖安装
- 确认数据格式正确
- 重启程序重新初始化
- 检查内存和性能
```
